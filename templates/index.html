Heart Disease Prediction System

About This Project

This project was developed by Sandeep Kumar as a final-year academic project. It demonstrates how machine learning techniques can be applied to healthcare data to help with early screening of heart disease.

Heart disease is one of the leading causes of death worldwide. Early detection is crucial for effective treatment and improved patient outcomes. This application uses a trained classification model to estimate the probability of heart disease based on 13 clinical parameters.

How It Works

  1. Learn about the model — Visit the Model page to see which algorithm was chosen and how it was evaluated.
  2. Understand the parameters — The Parameters page explains each of the 13 clinical inputs, with normal ranges and examples.
  3. Make a prediction — Go to the Predict page, fill in the patient data, and get a probability score.
  4. Review past results — The History page stores all past predictions in a database for reference.

Dataset

The model was trained on the Cleveland Heart Disease dataset from the UCI Machine Learning Repository.

Source UCI Machine Learning Repository
Records 303 patient records
Features 13 clinical attributes
Target Binary — heart disease vs. no heart disease
Contributors Hungarian Institute of Cardiology, University Hospital Zurich, University Hospital Basel, V.A. Medical Center Long Beach
